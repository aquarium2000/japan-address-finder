# japan-address-finder
指定した座標（緯度・経度）から日本の住所を取得するプログラムです。（国土地理院の無料データを使用）

install_map.sh wgetで地図データを取得して、演算しやすいように加工し、mapフォルダーを作成、その中に、細かく分割した座標情報のファイルを生成します。
map_make.py　　install_map.shで使用します。
map_rtable.py　距離計算を高速に行うためにテーブルを使いますが、そのテーブルの演算を行うプログラムです。map.pyで使用するデータを生成しますが、生成後のデータはmap.pyに含まれていますので、使用はしません。
map.py　　　　　緯度と経度を入力すれば近い箇所の住所が表示されます。（距離計算は正確ではありませんので大まかな住所とお考え下さい。）
